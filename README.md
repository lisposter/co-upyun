# co-upyun
A upyun sdk for node.js, which is for co-like interface. Can be used with `koa` or `co`.

# INSTALL
```sh
$ npm install co-upyun
```

# Usage
Please visit [wiki](https://github.com/lisposter/co-upyun/wiki) for temporary.
